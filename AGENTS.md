# Project Working Agreement

- Work autonomously for normal investigation, implementation, formatting, and
  verification within this repository.
- Do not ask for confirmation before reading or editing repository files,
  running local tests, or using non-destructive development commands.
- Preserve unrelated user changes and untracked files.
- Request approval before destructive operations, network access, or writes
  outside this repository.
- Complete implementation and verification when feasible instead of stopping
  at a proposal.
